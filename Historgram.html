User Guide

  1. Enter your data points. Separate values with commas.
  2. Specify a custom maximum Y-axis value.
  3. Enter a title for your graph.
  4. Provide labels for the Y-axis and X-axis.
  5. Define the range for the X-axis. (e.g., 0,10).
  6. Enter the number of bins (bars) you want for the histogram. This does need to be a multiple of the max X-axis value to display properly.
  7. Click the "Generate Histogram" button to create your histogram.

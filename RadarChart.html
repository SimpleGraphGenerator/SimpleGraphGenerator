Simple Radar Chart Generator

User Guide:

Here's how to use the RadarChart Generator:

  1. Enter a title for your radar Chart.
  2. Enter data names and values. To add more data fields, click the "Add New Field" button.
  3. Click the "Generate RadarChart" button to create your radar chart.
  4. Copy your graph to the clipboard, or download it as a JPEG.

Remember to input an equal number of labels and data values for the graph it render properly.
